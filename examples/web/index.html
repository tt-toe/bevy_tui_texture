bevy_tui_texture - WASM Demo

Bevy + ratatui running in your browser via WebAssembly!

About This Demo

This demonstrates bevy_tui_texture with SimpleTerminal3D running in the browser using WebAssembly and WebGL2.

  • Bevy 0.18 - Game engine running in browser
  • WGPU 27 - GPU acceleration via WebGL2
  • ratatui 0.30 - Terminal UI rendering
  • SimpleTerminal3D - 3D terminal demonstration

Built with Bevy + ratatui + WGPU
